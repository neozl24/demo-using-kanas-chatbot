This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Local Server
Before you can chat with AI, you need to prepare your local server which provides a chat endpoint.
See more info on [Kanas Chatbot](https://github.com/neozl24/kanas-chatbot) or [Server Demo page](https://github.com/neozl24/chatbot-server-demo)
